## System dependencies

This project uses [gitflow](https://github.com/nvie/gitflow).

## Configuration

Run `git flow init` to get into development.

Create `.env` file with the following keys and values for authentication in development environment:
```yaml
FACEBOOK_KEY: ...
FACEBOOK_SECRET: ...
```
